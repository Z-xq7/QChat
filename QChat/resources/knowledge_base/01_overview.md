# QChat 项目概述

## 项目简介

QChat 是一个基于 C++ 和 Node.js 的仿微信分布式即时通信系统，提供完整的聊天、好友管理、消息传递等功能。该项目采用分布式架构设计，包含多个服务器组件和一个跨平台客户端。

## 核心特性

1. **分布式架构**：采用微服务设计，各组件职责明确，便于扩展和维护
2. **高并发处理**：使用异步 I/O 和线程池处理大量用户连接
3. **实时通信**：TCP 长连接确保消息实时传递
4. **安全机制**：Token 验证、异地登录检测
5. **负载均衡**：根据服务器负载动态分配用户连接
6. **跨平台**：客户端基于 Qt，支持多平台部署
7. **用户友好**：界面设计仿微信，用户体验良好

## 技术栈

### 后端技术
- **编程语言**：C++ (服务器端), JavaScript/Node.js (验证服务)
- **网络库**：Boost.Asio (C++ 异步网络), gRPC
- **数据库**：MySQL (持久化存储), Redis (缓存、会话管理)
- **协议**：HTTP, TCP, gRPC
- **序列化**：Protocol Buffers, JSON

### 前端技术
- **GUI 框架**：Qt 6 / C++
- **UI 设计**：Qt Widgets, QSS 样式表
- **网络通信**：Qt Network

## 核心功能模块

### 用户系统
- 用户注册（邮箱验证）
- 用户登录/登出
- 密码重置
- 个人信息管理

### 好友系统
- 搜索用户
- 添加好友申请
- 好友申请审核
- 好友列表管理
- 好友信息查看

### 消息系统
- 实时文本消息传递
- 消息状态追踪（发送/已读）
- 聊天记录加载
- 消息时间戳显示
- 历史消息查询

### 群聊系统
- 创建群聊
- 群成员管理
- 群公告发布
- 群消息收发

### 视频通话系统
- P2P 视频通话（基于 metartc7）
- WebRTC 信令交换
- TURN 服务器支持

### AI 助手系统
- 基于大模型的智能对话

- 长期记忆功能（用户偏好、习惯）

- 知识库检索（RAG）

- 工具调用能力

- 语音输入功能

- 使用指南：

  - 发送消息（快捷键）：

    |       按键        |   功能   |
    | :---------------: | :------: |
    |     **Enter**     | 发送消息 |
    | **Shift + Enter** |   换行   |
    |   点击发送按钮    | 发送消息 |
