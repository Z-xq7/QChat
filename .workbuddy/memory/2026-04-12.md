## 知识库三连 Bug（全部修复 ✅）

| Bug | 根因 | 修复 |
|---|---|---|
| 刷新后全空 | 两 KB 共用 index.json + updateDocument 先删后找路径丢失 | 分离存储路径 + 先存 filePath 再 remove |
| TF-IDF 中文无效 | 字符 bigram 相似度仅 0.07~0.09, 远低于阈值 | 重写为关键词 n-gram 匹配 |
| 语义鸿沟漏检 | "详细信息" vs "姓名/年龄" 词汇不匹配 | 小KB(≤10chunks)兜底: 全部返回让AI判断 |

**最终方案**: 关键词匹配 → 单字回退 → 全量兜底(小KB)
**MinGW 注意**: 不支持 `{a,b}` 大括号初始化, `qMakePair` 类型顺序需注意

## 聊天列表时间显示功能（模仿微信）

### 需求
在 `chatdialog` 的聊天列表中，每个聊天 item 的时间显示最后一条消息的时间，模仿微信格式：
- 当天：显示具体几点几分（如"14:30"）
- 昨天：显示"昨天"
- 这周内：显示周几（如"周一"、"周三"）
- 更早：显示几月几日（如"5月12日"）

**服务器端（ChatServer）：**
- `data.h`: `ChatThreadInfo` 添加 `_last_msg_time` 字段（std::string 类型）
- `MysqlDao.cpp`: LEFT JOIN chat_message 获取 MAX(updated_at)，按时间降序排序
- `LogicSystem.cpp`: 返回 JSON 时添加 `last_msg_time` 字段

**客户端（QChat）：**
- `userdata.h`:
